# Direction - 動画制作・編集 業務効率化システム ガイド

**対象:** 動画制作・編集担当者
**最終更新:** 2026年3月31日

---

## このシステムでできること（全体像）

Directionは、動画制作の「面倒な作業」をAIが自動でやってくれるシステムです。
あなたは**クリエイティブな判断**に集中できるようになります。

```
あなたがやること          AIが自動でやること
─────────────          ─────────────────
指示を出す         →    素材の整理・分類・名称付け
確認・微調整する    →    無音部分のカット提案
イメージを伝える    →    参考動画を探して候補を提示
選ぶ・直す         →    テロップの自動生成・配置
イメージを伝える    →    BGM・SEの候補を提示
キーワードを伝える  →    過去素材から該当シーンを検索
何もしない         →    夜間に自動でファイルをダウンロード
```

---

## 8つの機能（Module）の Before / After

---

### Module 1: 素材の整理・分類

**何をしてくれるか:**
撮影した大量の動画素材を、AIが中身を見て自動的にシーンごとに分けて名前をつけます。

| | Before（今まで） | After（導入後） |
|---|---|---|
| 作業内容 | 動画を1本ずつ再生して確認し、手動でフォルダ分け・名前変更 | AIが動画の内容を分析して自動でシーン分割・名称付け |
| かかる時間 | 素材100本で **3〜5時間** | 素材100本で **15〜30分**（自動処理+確認のみ） |
| 精度 | 担当者の判断に依存 | カメラの画角・背景・シーン変化をAIが統一基準で分類 |

**あなたがやること:**
1. Google Driveに撮影素材をアップロード（今まで通り）
2. Claude Codeで `/素材整理 [フォルダパス]` と入力
3. AIが分類した結果を確認し、必要なら修正

---

### Module 2: 不要部分の自動カット

**何をしてくれるか:**
動画の無音部分や、裏方スタッフの声が入っているシーンを自動で検出してカット候補を作ります。

| | Before（今まで） | After（導入後） |
|---|---|---|
| 無音カット | タイムラインで波形を見ながら手動でカット | AIが無音2秒以上を自動検出→Premiere Proのタイムラインに反映 |
| スタッフ声 | 全編を聞きながら手動でチェック | AIが「誰の声か」を自動判別→スタッフの声をマーク |
| 出力先 | - | **Premiere Pro** のタイムラインとして直接出力 |

**あなたがやること:**
1. Claude Codeで `/カット [動画パス]` と入力
2. Premiere Proが開き、カット候補がマーク済みのタイムラインが表示される
3. マークされた箇所を確認し、必要なら手動で微調整
4. **最終的なカット判断は常にあなたが行います**

**補足 - スタッフ声の除去について:**
- 出演者の声を事前に登録しておくと、それ以外の声（=スタッフ）を自動識別できます
- ただし出演者とスタッフが同時に話している箇所は自動分離が難しいです
- ピンマイク収録と併用すると最も効果的です

### Module 3: 参考動画のリサーチ・録画・解析

**何をしてくれるか:**
3つの機能があります。

#### 3-A: 参考動画を探す

| | Before（今まで） | After（導入後） |
|---|---|---|
| 探し方 | TVer・YouTubeで手動検索、1つずつ視聴して確認 | イメージをAIに伝えるだけで、YouTube・Vimeo等から候補を自動検索 |
| 時間 | 探したいシーン1つにつき **30分〜1時間** | **5分以内** に候補リスト提示 |

**あなたがやること:**
1. Claude Codeで「料理番組風の、手元アップのシーンの参考がほしい」と伝える
2. AIがイメージを整理し、該当する動画を検索して候補リストを表示
3. 気に入ったものを選ぶ

#### 3-B: TVer/ABEMAの自動画面録画

| | Before（今まで） | After（導入後） |
|---|---|---|
| 記録方法 | 視聴しながらメモを取る程度 | 自動で画面録画→保存→シーン分割→タグ付けまで自動 |

**あなたがやること:**
1. Claude Codeで `/参考録画 [番組URL]` と入力
2. 自動でブラウザが開き、録画が始まる（PCは録画中専有）
3. 録画完了後、自動でシーン分割・タグ付けされてライブラリに蓄積

**注意:** この機能は個人参考用途のみです。録画データは外部に共有しないでください。

#### 3-C: 蓄積した参考動画から検索

| | Before（今まで） | After（導入後） |
|---|---|---|
| 検索方法 | 「前に見たあの番組のあのシーン」を記憶頼みで探す | AIに「テロップがポップで手元アップの料理シーン」と伝えれば即座に候補表示 |

**あなたがやること:**
1. Claude Codeで `/参考ライブラリ "探したいシーンの説明"` と入力
2. 蓄積されたライブラリから、3つの方法で検索:
   - セリフ（文字起こし）からキーワード検索
   - タグ（カメラ/雰囲気/テロップスタイル等）でフィルタ
   - 映像の「見た目」が似ているシーンをAIが検索
3. サムネイル付きで候補が表示される

---

### Module 4: テロップの自動生成

**何をしてくれるか:**
動画の文字起こしから、シーンに合ったテロップを自動で作成してPremiere Proに配置します。

| | Before（今まで） | After（導入後） |
|---|---|---|
| 文字起こし | 動画を聞きながら手動で書き起こし | AIが自動で高精度な日本語文字起こし（TV音声学習済み） |
| テロップ案 | 編集者がセンスで考える | AIが複数のテロップ候補を提案（バラエティ風/ニュース風等） |
| テロップ配置 | Premiere Proで1つずつ手動作成・配置 | 選んだテロップスタイルでPremiere Proに自動配置 |
| 修正 | - | Premiere Pro上で通常通り自由に修正可能 |
| 追加料金 | - | **なし**（既存のAdobe CC契約内） |

**あなたがやること:**
1. Claude Codeで動画のコンセプトやトーンを伝えて壁打ち
   - 「バラエティ風でポップなテロップにしたい」
   - 「ドキュメンタリー風で控えめに」
2. AIがテロップ候補を提案 → 気に入ったスタイルを選択
3. Premiere Proにテロップが自動配置される
4. **細かい修正はPremiere Pro上で通常通り行えます**

---

### Module 5: BGM・SE のリサーチ

**何をしてくれるか:**
欲しい音のイメージを伝えるだけで、複数の音源サイトから候補を自動で探して提示します。

| | Before（今まで） | After（導入後） |
|---|---|---|
| 検索方法 | MotionArray・Artlistで手動検索、1つずつ試聴 | AIにイメージを伝えるだけで複数サイトを横断検索 |
| 検索対象 | 契約中の2サイトのみ | 契約サイト + 無料音源サイト + AI生成も含めて横断検索 |
| 時間 | 1シーンのBGM探しに **30分〜1時間** | **5分以内** に候補リスト提示 |

**あなたがやること:**
1. Claude Codeで「感動的なピアノ曲、ゆっくりめ、ボーカルなし、1分半くらい」と伝える
2. AIが以下のサイトから候補を自動検索:
   - Epidemic Sound（契約時）
   - Freesound.org（無料）
   - Jamendo（無料・商用可）
   - AI生成音楽（著作権フリー）
3. プレビュー付きの候補リストから選ぶ

**注意:** MotionArrayは利用規約でAPI自動検索が禁止されているため、従来通り手動検索になります。

### Module 6: 過去動画素材の掘り起こし

**何をしてくれるか:**
Google Driveに保存されている全ての過去動画素材を自動で文字起こしして検索可能にします。

| | Before（今まで） | After（導入後） |
|---|---|---|
| 素材の探し方 | フォルダを1つずつ開いて再生して確認 | Claude Codeにキーワードやイメージを伝えるだけ |
| 整理状態 | 撮影日・カメラ番号程度の大まかな分類 | 全動画の文字起こし + シーンごとのタグ付け + ベクトル検索 |
| 検索時間 | 目的のシーンを探すのに **数時間** かかることも | **数秒〜数分** で候補を表示 |

**あなたがやること:**
1. 素材はこれまで通りGoogle Driveにアップロード
2. **自動で**文字起こしファイルが生成され、同じフォルダに保存される
3. 探したいときは Claude Codeで `/素材検索 "田中さんが商品を説明しているシーン"` と入力
4. 該当する動画名・タイムスタンプ・サムネイルが表示される

---

### Module 7: 動画素材の自動ダウンロード

**何をしてくれるか:**
Google Driveの動画素材を、夜間に自動でローカルPCにダウンロードします。

| | Before（今まで） | After（導入後） |
|---|---|---|
| ダウンロード方法 | ブラウザから手動で1つずつダウンロード | 夜間に全自動で同期ダウンロード |
| 待ち時間 | 大容量ファイルのDL完了を待つ必要あり | 朝出勤したら完了済み |
| 操作 | 毎回ファイルを選んでDLクリック | **完全自動（設定後は何もしなくてOK）** |

**あなたがやること:**
- **初回のみ:** ダウンロードしたいGoogle Driveのフォルダと、保存先のローカルフォルダを設定
- **以降:** 何もしなくてOK。毎晩自動で新しいファイルが同期されます
- 完了通知が届くので、朝確認するだけです

## 大切なポイント

### AIがやること / あなたがやること

| AIが自動でやること | あなたが判断すること |
|---|---|
| 動画の分析・分類・名称付け | 分類結果の最終確認 |
| 無音部分・不要シーンの検出 | **カットするかどうかの最終判断** |
| 参考動画の検索・候補提示 | どの参考を採用するか |
| テロップの文字起こし・候補作成・配置 | テロップの最終的な表現・修正 |
| BGM・SEの候補検索 | どの曲を使うかの選択 |
| 過去素材の検索・候補表示 | どの素材を使うかの判断 |
| ファイルのダウンロード | - |

**要するに:**
- **AI = 面倒な作業を自動化する「超高速アシスタント」**
- **あなた = クリエイティブな判断を行う「最終決定者」**
- AIが出した結果は、全てあなたがPremiere Pro上で自由に修正できます

### 今の作業環境はそのまま

- **Premiere Pro** をそのまま使い続けます（追加料金なし）
- **Google Drive** の使い方も変わりません
- **MotionArray / Artlist** もそのまま併用できます
- 新しいソフトを覚える必要はほとんどありません
- Claude Code（チャット画面）に指示を出すだけで各機能が動きます

## よくある質問

**Q: 新しいソフトを覚える必要がありますか？**
A: ほぼありません。Premiere Proはそのまま使えます。Claude Code（チャット画面）に日本語で指示を出すだけです。

**Q: AIの判断が間違っていたらどうなりますか？**
A: AIはあくまで「候補の提案」をするだけです。最終判断は全てあなたが行います。カット・テロップ・BGMなど、全てPremiere Pro上で自由に修正できます。

**Q: インターネット接続は必要ですか？**
A: はい。Google Drive連携やAI処理にインターネット接続が必要です。

**Q: 既存の編集作業に影響はありますか？**
A: ありません。既存のプロジェクトファイルや素材には一切手を加えません。新しい機能を「追加」するだけです。

**Q: 一度に全部使わないといけませんか？**
A: いいえ。8つのModuleは独立しているので、必要なものだけ使えます。段階的に導入していきます。
